## PU-01

**PU-01** was designed by me in the class of **"Fundamentals of Unmanned Aerial Vehicle System Design"** back in 2016.

It is a twin boom fixed wing aircraft with a pusher setup and can be controlled manually by human on ground.

<img src="https://github.com/shengwen1997/PU-01/blob/master/materials/pu-01.jpg?raw=true" width="75%" height="75%"></a>

## Specification

| Parameter | |
|:-------------------|:---------|
| Cruise speed       | 52km/hr  |
| Max take-of weight | 1.2kg    |
| Main wing airfoil  | Clark-Y  |
| Main wing span     | 1.13m    |
| Tail wing airfoil  | NACA0015 |
| Motor              | 1400kv   |
| Propeller          | 8060     |
